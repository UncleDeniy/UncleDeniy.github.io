# Запущен единый центр статей и новостей

> [!NOTE]
> В SysHelper появился единый knowledge hub: теперь можно вести **статьи**, **новости**, **обновления проекта** и открывать всё это через удобный markdown-reader.

## Что появилось

- отдельная витрина для материалов;
- фильтрация по типу контента;
- красивые карточки;
- единый просмотрщик markdown;
- импорт локальных `.md` файлов;
- поддержка callout-блоков, таблиц, кода и изображений.

## Для чего это удобно

Теперь SysHelper можно использовать не только как набор страниц, но и как:

- личную IT-вики;
- журнал обновлений;
- ленту внутренних новостей;
- архив инструкций и заметок.

## Что дальше логично развивать

| Направление | Что можно добавить |
|---|---|
| Новости | закреплённые посты, важность, обложки |
| Статьи | связанные материалы, избранное, серия статей |
| Поиск | быстрый глобальный поиск по всем разделам |
| Экспорт | печать статьи и экспорт в PDF |

> [!TIP]
> Для новостей удобно использовать единый шаблон: дата, краткое описание, изменения, что делать дальше.

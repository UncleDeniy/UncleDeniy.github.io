Структура архива:
- index.html — главная страница
- styles.css — стили
- script.js — логика тем и галереи
- assets/ — иконка и превью проектов

Темы:
- Auto
- Light
- Dark
Переключаются кнопкой в шапке.

Чтобы заменить превью проектов на свои скриншоты:
1. Открой папку assets
2. Замени файлы uncle.svg, phobos.svg, duga.svg, weather.svg, syshelper.svg, galaxy.svg
3. Лучше использовать изображения с таким же именем и соотношением сторон около 16:9
4. Можно использовать .jpg/.png/.webp — тогда поправь путь в index.html

Внутренние ссылки на проекты сейчас ведут на:
- ./tayna-stancii-duga/index.html
- ./weathervision/index.html
- ./syshelper/index.html
- ./galaxy-defender/index.html
Если папки будут называться иначе — измени пути в index.html

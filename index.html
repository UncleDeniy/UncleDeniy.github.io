Арсенал разработчика на Arch Linux

Установите всё необходимое для комфортной разработки в несколько команд. Эффективные инструменты для профессионалов.

Начать установку

Веб-серверы

Apache HTTP Server Official

Мощный и гибкий веб-сервер с широкой поддержкой модулей.

sudo pacman -S apache

Офисные приложения

OnlyOffice AUR

Мощный офисный пакет с поддержкой форматов Microsoft Office.

yay -S onlyoffice-desktopeditors

LibreOffice Official

Мощный офисный пакет с открытым исходным кодом.

sudo pacman -S libreoffice-fresh

Системные утилиты

VirtualBox Official

Мощная система виртуализации для корпоративного и домашнего использования.

Динамический тайловый оконный менеджер на основе wlroots.

Установка AUR helpers

Yay Official

Популярный помощник для установки пакетов из AUR.

# Установка зависимостей
sudo pacman -S --needed git base-devel

# Клонирование репозитория yay
git clone https://aur.archlinux.org/yay.git

# Переход в директорию с yay
cd yay

# Сборка и установка пакета
makepkg -si

Paru Official

Форк yay с дополнительными функциями и улучшениями.

# Установка через pacman (есть в официальных репозиториях)
sudo pacman -S paru
Команда скопирована в буфер обмена!
